# Safety Boundary

This package does not implement strike, weapon release, collision attack, or destructive target engagement.

Allowed behavior in this package:

- swarm formation
- communication health
- collision avoidance
- point-A to point-B navigation
- camera/object inspection
- non-destructive object confirmation/tagging

Not included:

- terminal impact guidance
- payload release
- motor kill of another vehicle
- target destruction logic
- code intended to harm people, vehicles, or property
